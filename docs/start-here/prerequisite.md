# Prerequisite

## Foundry Installation

```shell
curl -L https://foundry.paradigm.xyz | bash
source ~/.zshrc
foundryup
```

## Node Installation

```shell
brew install nvm # follow https://formulae.brew.sh/formula/nvm
nvm install 14
nvm use 14
```

# Installing Dependencies

install node modules

```shell
npm install --force
```

install foundry libraries
```shell
forge install
```
