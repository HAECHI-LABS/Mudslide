# Instructions

## use template using foundry

```shell
forge init --template https://github.com/haechi-labs/mudslide <TARGET_FOLDER>
```

## using foundry

### running compile

```shell
npm run compile
```

### running test

```shell
npm run test
```

## using hardhat

### running compile

```shell
npm run compile:hardhat
```

### running test

```shell
npm run test:hardhat
```
